# Model attribution

This project uses the following upstream model and resources:

- Base model: HuggingFaceTB/SmolVLM-256M-Instruct
  - https://huggingface.co/HuggingFaceTB/SmolVLM-256M-Instruct
- Fine-tuning reference notebook:
  - https://github.com/huggingface/smollm/blob/main/vision/finetuning/Smol_VLM_FT.ipynb

Please review and comply with the model card, license, and usage terms before redistribution of fine-tuned checkpoints.
